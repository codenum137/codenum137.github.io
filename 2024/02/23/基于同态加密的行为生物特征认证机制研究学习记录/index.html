基于同态加密的行为生物特征认证机制研究学习记录

文献

AuthentiSense: A Scalable Behavioral Biometrics Authentication Scheme using Few-Shot Learning for Mobile Platforms.

dblp: AuthentiSense: A Scalable Behavioral Biometrics Authentication Scheme using Few-Shot Learning for Mobile Platforms.

摘录

  • 传统认证（指纹，脸部，虹膜）不安全，可被窃取复制，且不保护对底层设备的访问权限。
  • 行为生物特征认证机制通过用户持续的与移动设备交互的特征来认证，但目前的认证机制有两点不足：
    • 用户有关性，即不能在无预训练的前提下进行认证。
    • 不能进行大规模的用户认证。
  • AuthentiSense（以下简称AS）是一个用户无关，高效的可大规模认证机制。
  • AS利用少样本（3次样本）学习技术，也称孪生网络，来实现大规模和用户无关性。

引言

  • 这篇论文中，我们利用的生物行为特性是motion sensor（加速度）。
  • 现存生物特征认证的缺陷：
    • 用户有关性，需要大量训练数据来构建模型。
    • 不可规模认证，仅能认证限制数量的。
    • 每个用户都需要一个模型，这很需要资源。
    • 需要长时间的交互来认证。
  • AS的目的：
    • 高效，不需要特征提取（人工）的模型训练。
    • 可规模化认证。
    • 用户无关性，即用户动态变化时，不需要重训练。
  • AS利用了少样本学习技术，即孪生神经网络。该技术已经被用于现有的生物特性认证，但并未解决规模化认证的问题。
  • AS的贡献可总结如下
    • 用户无关，大规模认证，少样本学习，不需要大量训练数据。
    • 建立了端到端的神经网络架构，能够在不重训练的前提下动态处理用户变化。有97%的精准度，能1s完成认证。
    • 提供了一种评估不同参数影响的可扩展的系统测量方法。

背景知识

  • 少样本学习
  • 指用尽量少的训练数据来生成足够精确的模型。
  • 孪生网络
    • 一个孪生网络是两个给定的认证网络架构的串联，两个子网络共享相同的参数（权重和偏差），目的是为了在两个输入样本属于同类时提取相似特征，不属于同类时提取差异特征。

系统和威胁模型

  • 我们作出以下假设：
    • 认证模型被服务器训练和保存，该服务器是可信任的，这也避免了模型污染。
    • 攻击方能够威胁用户的移动设备。
    • 传感器数据是可信任的，被硬件安全技术（ARM TrustZone）所保护。同样，生物认证模块也是被硬件安全技术保护。这意味着即使OS被破坏，攻击方仍无法获取传感器数据。
    • 不考虑侧道攻击（side-channel attack)。
    • 传输信道是安全的，由传输安全协议保护，如SSL/TLS。故中间人攻击和重放攻击是无效的。
  • 为了认证一个用户，AS会收集3个数量的注册样本。在初始化后，AS会比较注册样本和用户样本（这里注册样本enrollment理解为认证样本更好，用户样本指的应该是之前保存的样本）。如果两者属于同一个用户，AS会生成内嵌向量，令生成空间中两者的欧氏距离减少，AS还会生成二进制标志‘1’，表示成功。
  • 当AS生物特征认证失败后，会采用消极认证方式（如密码）来保证安全。

AuthentiSense

总览

  • 孪生网络：共享权重的子网络，处理输入的用户样本。
  • 距离函数：计算欧氏距离，可看做向量，增大不同用户的样本的向量距离，减少相同用户的样本所生成的向量距离。
  • 决策网络：由全连接层组成，进行二元分类。样本属于相同用户则生成1，不同则生成0。
设计
  • 孪生网络：用于提取判别特征。令相同用户样本生成的向量靠近，不同用户样本生成的向量分离。由两个相同的子网络（CNN1D）组成，
