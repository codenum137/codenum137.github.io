dockerfile

  • build

    docker build -t authserver_ubuntu:v3 .

  • run，--network=host是联网功能。

    docker run -it --network=host authserver_ubuntu:v3

  • 第一次尝试编写的dockerfile，重点在于下载libmysqlclient的库，运行时需要加&，让程序后台运行，才能build成功。

    FROM ubuntu
MAINTAINER hey <3261575578@qq.com>
RUN sed -i s@/archive.ubuntu.com/@/mirrors.aliyun.com/@g /etc/apt/sources.list
RUN sed -i s@/security.debian.org/@/mirrors.aliyun.com/@g /etc/apt/sources.list
RUN sed -i s@/deb.debian.org/@/mirrors.aliyun.com/@g /etc/apt/sources.list
RUN sed -i s@/snapshot.debian.org/@/mirrors.aliyun.com/@g /etc/apt/sources.list
RUN apt-get update
RUN apt-get install g++ -y
RUN apt-get install libmysqlclient-dev -y
RUN mkdir /AuthServer
COPY . /AuthServer
WORKDIR /AuthServer
RUN /usr/bin/g++ -g main.cpp -o main -lpthread -I /AuthServer/include /AuthServer/res/*.cpp `mysql_config --cflags --libs`
RUN ./main &

  • dockerfile的编写可以去run你的基础映像，在里面尝试将代码编译好，然后同样步骤写到dockerfile中，但需要注意build有时无法联网的问题，本人通过修改dns得到解决。下面网站提到可根据docker版本同样添加--network=host，未来可尝试一下。

扩容

  1. 确保路由器接入公网，即与NAT相连。

  2. 右键路由器->configure->HDD->Resize，注意resize直接是increase

  3. 参考以下网站解决。

    Debian Linux磁盘扩容 - Jrri - 博客园 (cnblogs.com)

  4. install时出错：unable to parse xxx可参考以下网站解决。

    [Solved] Unable to parse package file /var/lib/apt/lists (itsfoss.com)

配置静态路由

up ip route add 192.168.2.0/24 via 192.168.0.2

配置ip和默认网关

  1. 执行下面指令

    sudo -s
vim /etc/network/interfaces
# 注释掉 "source-directory /run/network/interfaces.d" 这一行
rm /etc/network/interfaces.d/50-cloud-init
# 创建 "/etc/network/interfaces.d/10-ens4"
vim /etc/network/interfaces.d/10-ens4

  2. 添加内容,保存并退出编辑器

    auto ens4
iface ens4 inet6 static
        address 2023:10::1/64
        gateway 2023:10::11 #注意此处不能有/64，否则sudo systemctl restart networking会失败
iface ens4 inet static
        address 10.10.10.11/24
up ip route add 10.10.0.0/16 via 10.10.10.21

  3. 重新启动网络服务使更改生效

    sudo systemctl restart networking

  4. 查看是否生效

    ip addr

增加网络接口

  1. 首先增加网络适配器接口数量，右键config——adapter设置为2，用cloud/NAT（GNS3 VM）连接

  2. 编辑/etc/network/interfaces

    sudo vim /etc/network/interfaces

    添加内容

    auto ens5
iface ens5 inet dhcp

  3. 重新启动网络服务使更改生效

    sudo systemctl restart networking

增加网络接口需注意默认网关是如何设置的，本项目中采取默认网关设为外网接口，配置静态路由将内网ip汇聚到内网接口上。

sonic路由

isis开机自启动脚本

  1. 创建脚本

    vim docker_isisd.sh

    脚本内容如下

    cat << EOF > start_isisd.service
[Unit]
Description=Docker ISISD Service
After=docker.service
Requires=docker.service

[Service]
ExecStart=/usr/bin/docker exec -i bgp /bin/bash -c "/usr/lib/frr/isisd -A 127.0.0.1 -d"
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
EOF
sudo mv start_isisd.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable start_isisd.service

    here文档

    如果你需要输入多行文本，可以使用Here文档（Here Document）来实现。Here文档允许你在脚本中直接定义多行文本，并将其写入文件。

    下面是一个示例脚本，演示如何新建一个文件并输入多行文本：

    复制代码#!/bin/bash

# 创建一个新文件并输入多行文本
cat << EOF > myfile.txt
Hello, World!
This is line 2.
And this is line 3.
EOF

    在上面的示例中，使用了cat命令和Here文档的语法。<< EOF表示开始定义Here文档，EOF表示结束Here文档。在这之间的内容就是要输入的多行文本。这里定义了三行文本，并将其写入到名为myfile.txt的文件中。

    如果脚本执行成功，将会在当前目录下创建一个新的文件myfile.txt，其中包含三行文本。

    请注意，Here文档中的EOF可以是任意标识符，只需保证开始和结束的标识符一致即可。

  2. 增加权限

    sudo chmod +x docker_isisd.sh

  3. 执行脚本

    ./docker_isisd.sh

自动化配置ip，isis

  • 利用sed -i命令去编辑interfaces文件

    注释

    sed -i -e 's/source-directory \/run/# source-directory \/run/g' /etc/network/interfaces

    添加内容，在第8行后添加

    sed -i -e '8a 
auto ens5
iface ens5 inet dhcp' /etc/network/interfaces

  • 脚本文件中使用sudo -s会中断脚本，故直接sudo 命令比较好。

  • sudo cat << EOF > 10-ens4无法在权限文件夹创建10-ens4，故需要修改上级目录的权限

  • vtysh无法通过echo -e “xxx” | vtysh和here文档来交互，但可以通过-c命令来解决。

部署步骤：

  1. 创建脚本后复制对应脚本内容。

    vim init.sh

  2. 增加权限。

    sudo chmod +x init.sh

  3. 运行脚本。

    ./init.sh

# 配置isis自启动
# 固定不变的
# 配置ip
#     针对每个接口，需要以下命令
#     sudo config interface ip add Ethernet0 10.str(rid).str(oid).str(rid or oid) /8
#     sudo config interface startup Ethernet0
#     所有接口配置完需要保存
#     sudo config save
# 配置isis(在isisd服务配置好的情况下)
#     vtysh
#     config
#     ro isis sonic
#     net 49.0001.0000.0000.str(rid)(左边补齐0).00
#     针对每个接口,需要以下命令
#     interface Ethernet0
#     ipv6 ro isis sonic
#     ip ro isis sonic
#     exit
#     最后需要保存
#     end
#     write

ROUTE_NUM = int(input("请输入路由器数目\n"))
TERM_NUM = int(input("请输入终端数目\n"))


def make_ipv4(my_id, oid, mask):
    if my_id < oid:
        ip = "10." + str(my_id) + "." + str(oid) + "." + str(my_id)
    else:
        ip = "10." + str(oid) + "." + str(my_id) + "." + str(my_id)
    if mask > 0:
        ip = ip + "/" + str(mask)
    return ip


def make_ipv6(my_id, oid, mask):
    if my_id < oid:
        ip = "2023:" + str(my_id) + ":" + str(oid) + "::" + str(my_id)
    else:
        ip = "2023:" + str(oid) + ":" + str(my_id) + "::" + str(my_id)
    if mask > 0:
        ip = ip + "/" + str(mask)
    return ip


class Terminal:
    def __init__(self, tid):
        # tid:term_id
        self.oid = None
        self.id = tid + ROUTE_NUM

    def add_ens4_route(self, oid):
        self.oid = oid

    def make_sh(self):
        file_name = "t" + str(self.id - ROUTE_NUM) + ".sh"
        with open(file_name, "w") as f:
            pre_order = \
                """sudo sed -i -e 's/source-directory \\/run/# source-directory \\/run/g' /etc/network/interfaces
sudo sed -i -e '8a \\\\nauto ens5\\niface ens5 inet dhcp' /etc/network/interfaces
sudo rm /etc/network/interfaces.d/50-cloud-init
"""
            f.write(pre_order)
            # 修改权限
            chmod = "sudo chmod 777 /etc/network/interfaces.d\n"
            f.write(chmod)
            # 配置ip和默认网关
            my_ipv6 = make_ipv6(self.id, self.oid, 64)
            o_ipv6 = make_ipv6(self.oid, self.id, 0)
            ipv4 = make_ipv4(self.id, self.oid, 24)
            o_ipv4 = make_ipv4(self.oid, self.id, 0)
            ip_config = """sudo cat << EOF > /etc/network/interfaces.d/10-ens4
auto ens4
iface ens4 inet6 static
address """ + my_ipv6 + """
gateway """ + o_ipv6 + """
iface ens4 inet static
address """ + ipv4 + """
up ip route add 10.0.0.0/8 via """ + o_ipv4 + """
EOF
"""
            f.write(ip_config)
            restart_net = "sudo systemctl restart networking\n"
            f.write(restart_net)


class Router:
    def __init__(self, rid):
        # rid:router_id
        self.inter_map = &#123;&#125;
        self.id = rid

    def add_edge(self, _inter, oid):
        # oid:other_id
        self.inter_map[_inter] = oid

    def make_sh(self):
        file_name = "r" + str(self.id) + ".sh"
        with open(file_name, "w") as f:
            # 修改密码
            change_passwd = "echo -e \"123\n123\n\" | sudo passwd admin\n"
            f.write(change_passwd)
            # 配置isis自启动
            start_isis = """cat << EOF > start_isisd.service
[Unit]
Description=Docker ISISD Service
After=docker.service
Requires=docker.service

[Service]
ExecStart=/usr/bin/docker exec -i bgp /bin/bash -c "/usr/lib/frr/isisd -A 127.0.0.1 -d"
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
EOF
sudo mv start_isisd.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable start_isisd.service
/usr/bin/docker exec -i bgp /bin/bash -c "/usr/lib/frr/isisd -A 127.0.0.1 -d"
"""
            f.write(start_isis)
            # 配置ip
            for _inter, oid in self.inter_map.items():
                old_id = "10.0.0." + str(_inter * 2) + "/31"
                ipv4 = make_ipv4(self.id, oid, 24)
                ipv6 = make_ipv6(self.id, oid, 64)
                remove_ip = "sudo config interface ip remove Ethernet" + str(_inter * 4) + " " + old_id + "\n"
                add_ipv4 = "sudo config interface ip add Ethernet" + str(_inter * 4) + " " + ipv4 + "\n"
                add_ipv6 = "sudo config interface ip add Ethernet" + str(_inter * 4) + " " + ipv6 + "\n"
                start_up = "sudo config interface startup Ethernet" + str(_inter * 4) + "\n"
                f.write(remove_ip)
                f.write(add_ipv4)
                f.write(add_ipv6)
                f.write(start_up)
            save = "sudo config save -y\n"
            f.write(save)
            # 配置isis
            pre_order = """vtysh -c "
config
ro isis sonic
"""
            # net 49.0001.0000.0000.str(rid)(左边补齐0).00
            net = "net 49.0001.0000.0000." + "0" * (4 - len(str(self.id))) + str(self.id) + ".00\n"
            f.write(pre_order)
            f.write(net)
            for _inter in self.inter_map.keys():
                enter_inter = "interface Ethernet" + str(_inter * 4) + "\n"
                isis = "ipv6 ro isis sonic\nip ro isis sonic\n"
                f.write(enter_inter)
                f.write(isis)
                f.write("exit\n")
            f.write("end\n")
            f.write("write\"")


if __name__ == '__main__':
    router = [Router(0)]
    term = [Terminal(0)]
    for i in range(1, ROUTE_NUM + 1):
        router.append(Router(i))
    for i in range(1, TERM_NUM + 1):
        term.append(Terminal(i))
    print("请输入" + str(TERM_NUM) + "条终端链路信息 格式：（终端n通过ei接口与路由器a相连 n inter_ei aID")
    for i in range(TERM_NUM):
        edge = input()
        term_id, inter, r_id = edge.split()
        term_id = int(term_id)
        r_id = int(r_id)
        inter = int(inter)
        term[term_id].add_ens4_route(r_id)
        router[r_id].add_edge(inter, term[term_id].id)
    print("请输入路由链路信息，以空行结尾 格式：（路由器a的ei接口与路由器b的ej接口相连aID inter_ei inter_ej bID）")
    while True:
        edge = input()
        if edge == "":
            break
        s_router, s_inter, d_inter, d_router = edge.split()
        s_router = int(s_router)
        s_inter = int(s_inter)
        d_router = int(d_router)
        d_inter = int(d_inter)
        router[s_router].add_edge(s_inter, d_router)
        router[d_router].add_edge(d_inter, s_router)

    for i in range(1, ROUTE_NUM + 1):
        router[i].make_sh()
    for i in range(1, TERM_NUM + 1):
        term[i].make_sh()
